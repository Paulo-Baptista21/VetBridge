# VetBridge

Project focused on developing tools to support veterinary clinical routines.

## Current Features

- Medication dosage calculator

## How It Works

The calculator receives:

- Patient weight (kg)
- Recommended dosage (mg/kg)
- Drug concentration (mg/ml)

And returns:

- Total dose (mg)
- Administration volume (ml)

- Python
